We’re pleased to announce the availability of the long-awaited release: RethinkDB 2.4.0 (Night Of The Living Dead). This update includes a range of bug fixes, stability improvements, new features, and more. For a complete list of changes in this update, you can refer to the release notes . To learn more about what else going on around RethinkDB, please read our latest blog post . Here are some of the highlights of this release: Write hooks - attach to tables a function that can modify the behavior of any write Bitwise operations - allows basic bitwise operations such as AND, NOT, OR, XOR, SAL and SAR Permitted the hyphen character to be used in table names Improved table page performance in the case with many databases
